=== WP-Delinquify ===
Contributors: TheChef
Tags: link remove, anti-spam, no links

== Description ==

WP-Delinquify is a simple and light plugin that masks all the links from commnets replacing any http:// with hxxp://
and removing any html <a> tag
For more informations about pad files visit http://ezbitz.com/wp-delinquify/

== Installation ==

Upload WP-Delinquify plugin to your blog, Activate it, and that's it !
